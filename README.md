# fit
Home-based Bodyweight Exercise Program Generator Tool, powered by Node.

# Getting started 

#### Install dependencies:
> cd fit && npm install

#### On MacOS or Linux, run the app with this command:
> npm start
  
#### On Windows, use this command:
> npm start


# Test

> http://localhost:3000
